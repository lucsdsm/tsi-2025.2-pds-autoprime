AutoPrime - Sistema de Gerenciamento de Carros

1. Salvar Carro (saveCarro)


2. Consultar Preço (getCarro)


3. Atualizar Carro (updateCarro)


4. Deletar Carro (deleteCarro)


5. Listar Todos os Carros (listarCarros)

Instruções:

  • Para salvar: informe "modelo,preço" (ex: BMW,350000)
  • Para consultar: informe apenas o modelo (ex: BMW)
  • Para atualizar: informe "modelo,novo_preço" (ex: BMW,375000)
  • Para deletar: informe apenas o modelo (ex: BMW)
  • Para listar: apenas clique no botão

Nota: Os endpoints retornam JSON. Para testar via ferramentas como Postman ou curl, use:

  • POST http://localhost:8080/getCarro - body: modelo=BMW
  • POST http://localhost:8080/saveCarro - body: modelo=BMW,350000
  • POST http://localhost:8080/deleteCarro - body: modelo=BMW
  • POST http://localhost:8080/updateCarro - body: modelo=BMW,375000
  • POST http://localhost:8080/listarCarros - sem body
